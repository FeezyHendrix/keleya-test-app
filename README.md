
# Keleya Mobile App



 




## Installation

- Any Operating System (ie. MacOS X, Linux, Windows)
- Any IDE with Flutter SDK installed (ie. IntelliJ, Android Studio, VSCode etc)


```bash
  flutter pub get
```
    

App directory
- lib
  - screens
  - utils
  - widgets

Run on android
```
flutter run android
```

Run on ios

```
flutter run ios
```
